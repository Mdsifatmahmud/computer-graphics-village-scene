# 🌿 Dynamic Village – OpenGL Computer Graphics Project

**Dynamic Village** is an interactive **2D computer graphics simulation** developed using **C++ and OpenGL (GLUT)**. The project visualizes a lively rural village environment with animated natural elements such as moving clouds, flying birds, flowing rivers, boats, trees, and weather effects.

The scene dynamically changes between **Day Mode, Night Mode, and Rain Mode**, creating an immersive environmental experience. Each mode presents different visual behaviors and ambient sounds, demonstrating the use of **animation, transformations, and real-time rendering** in OpenGL.

This project was developed as part of the **Computer Graphics Laboratory course** at **Daffodil International University**.

---

## ✨ Key Features

• 🌤 **Day Mode** – Bright sky with moving clouds, birds, and boat animation
• 🌙 **Night Mode** – Dark sky with moon, stars, and calm nighttime environment
• 🌧 **Rain Mode** – Animated rainfall effect with environmental atmosphere
• ☁ **Dynamic Cloud Movement** using OpenGL transformations
• 🐦 **Flying Bird Animation** across the sky
• ⛵ **Boat Animation** moving along the river
• 🌳 Detailed **village environment** including trees, huts, wells, grass, and fences
• 🔊 **Environmental Sound Effects** for day, rain, and night modes
